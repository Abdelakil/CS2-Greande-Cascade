# Changelog

All notable changes to Grenade Cascade will be documented in this file.

## [1.0.0] - 2026-04-06

### Added
- Initial release of Grenade Cascade plugin
- Permission-based grenade cascade system
- Configurable grenade count spawning
- Recursion prevention to avoid infinite loops
- Natural grenade scattering with randomized velocities
- Full localization system with English translations
- Hot-reload configuration support
- Comprehensive logging and error handling
- Production-ready architecture following SwiftlyS2 standards

### Features
- HE grenade explosion event handling
- Permission checking system
- Entity tracking for spawned grenades
- Configurable plugin settings
- Multi-language support framework
- Memory-safe implementation

### Technical
- .NET 10.0 target framework
- SwiftlyS2 plugin architecture
- Async-safe operations
- Thread-safe entity handling
- Minimal performance overhead
